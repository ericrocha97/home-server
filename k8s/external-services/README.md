# External Services (Templates)

This directory does not store concrete manifests with real LAN IPs.

Rendered manifests are generated by Ansible from templates in:

- `ansible/templates/external-services/*.yaml.j2`

The IP used in `EndpointSlice.endpoints.addresses` comes from runtime variables:

1. `SERVER_LAN_IP` environment variable (preferred for public repo hygiene)
2. `ansible_host` from `ansible/inventory.ini` when `SERVER_LAN_IP` is not set

Example:

```bash
export SERVER_LAN_IP="<SERVER_LAN_IP>"
ansible-playbook -i ansible/inventory.ini ansible/k8s_apps_playbook.yml --ask-become-pass
```
